# Rock Paper Scissors Game

A simple **Rock Paper Scissors web game** built using **HTML, CSS, and JavaScript**.
Play against the computer and see who wins!

## Features

* Play Rock, Paper, Scissors against the computer
* Random computer move generation
* Score tracking
* Auto-play mode
* Simple and responsive UI

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/tejastiwari1456/rock-paper-scissors.git
```

2. Open the project folder

3. Run `rock-paper-scissors-original.html` in your browser
